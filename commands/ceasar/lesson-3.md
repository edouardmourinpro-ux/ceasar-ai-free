## Étape 3 — Guide d'installation personnalisé

Dis :

**Étape 3 — le guide concret. Comment installer ton premier MCP.**

Je vais te donner les étapes exactes pour le MCP le plus utile pour ton profil.

Affiche le guide d'installation pour le MCP #1 recommandé à l'étape 1.

Si c'est Notion, affiche :

```
  GUIDE D'INSTALLATION — NOTION MCP

  Étape 1 — Créer une intégration Notion
  → Va sur notion.so/my-integrations
  → Clique "Nouvelle intégration"
  → Donne-lui un nom (ex: "Claude Code")
  → Copie le token (commence par secret_)

  Étape 2 — Configurer Claude Code
  Ajoute ceci dans ~/.claude.json :

  {
    "mcpServers": {
      "notion": {
        "command": "npx",
        "args": ["-y", "@notionhq/notion-mcp-server"],
        "env": {
          "OPENAPI_MCP_HEADERS": "{\"Authorization\": \"Bearer TON_TOKEN\",
          \"Notion-Version\": \"2022-06-28\"}"
        }
      }
    }
  }

  Étape 3 — Connecter tes pages
  → Dans chaque page Notion : "..." → Connexions → ton intégration

  Étape 4 — Tester
  → Redémarre Claude Code
  → Tape : "Lis ma page Notion [nom] et résume-la"
```

Adapte le guide au MCP #1 recommandé si ce n'est pas Notion — même format, mêmes étapes claires.
