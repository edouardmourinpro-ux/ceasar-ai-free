## Étape 7 — Dernier cadeau : le Cheat Sheet

Crée le fichier `~/.claude/commands/ceasar/cheat-sheet.md` avec ce contenu :

```markdown
---
description: "Cheat Sheet Ceasar.ai — tout le cours en une page"
---

# CHEAT SHEET CEASAR.AI

## Commandes du cours
/ceasar:start       — Menu principal
/ceasar:lesson-1    — Setup & CLAUDE.md
/ceasar:lesson-2    — Automatisations
/ceasar:lesson-3    — Contenu
/ceasar:lesson-4    — MCPs
/ceasar:lesson-5    — Agents
/ceasar:lesson-6    — Build

## Skills gratuites
/ceasar:synthese    — Résume n'importe quel contenu
/ceasar:email-pro   — Email pro en 1 minute
/ceasar:hook-gen    — 10 hooks viraux
/ceasar:script-fr   — Script YouTube/Reel complet

## Fichiers clés
~/.claude/CLAUDE.md          — Mémoire globale de Claude
~/.claude/commands/          — Toutes tes skills
[ton-projet]/CLAUDE.md       — Mémoire du projet

## Installer un MCP (exemple Notion)
Ajouter dans ~/.claude.json :
{
  "mcpServers": {
    "notion": {
      "command": "npx",
      "args": ["-y", "@notionhq/notion-mcp-server"],
      "env": {
        "OPENAPI_MCP_HEADERS": "{\"Authorization\": \"Bearer TON_TOKEN\", \"Notion-Version\": \"2022-06-28\"}"
      }
    }
  }
}

## Lancer des agents en parallèle
Terminal 1 → claude → Agent 1
Terminal 2 → claude → Agent 2
Les agents communiquent via des fichiers partagés (.md)

## Communauté
skool.com/ceasar-ai
```
