## Règle fondamentale

Si Edouard envoie une vidéo, elle est **déjà virale et validée**. Ne change pas l'angle, le format, ni la structure. Ton seul job : l'adapter en français pour Ceasar.ai (Claude Code, agents IA, automatisation). Copie ce qui marche, change le contexte.

---

## Étape 2 — Analyse (interne, ne pas afficher)

Identifie :
- **Ce que montre la vidéo** : description visuelle précise — que voit-on à l'écran, que fait le créateur, quel est le CTA
- **Le hook** : les 3 premières secondes — qu'est-ce qui accroche ?
- **Le format** : durée, structure, nombre d'étapes
- **L'angle** : résultat choc / curiosité / douleur / transformation / insider
- **Le mécanisme** : pourquoi ça marche psychologiquement (curiosité, preuve sociale, engagement...)

---

## Étape 3 — Output

Livre en une seule réponse, sans demander de validation intermédiaire :

### 🎣 Hook adapté
La première phrase, prête à dire face caméra ou en voix over. Directe, en français parlé.

### 📋 Script complet

Adapte la structure selon le format de la vidéo source :

**REEL 30s :**
```
[HOOK — 3 sec] Phrase choc. Montre l'écran ou ton visage.
[DÉMO — 20 sec] 3 étapes max. Voix over. Montre ce qui se passe à l'écran.
[CTA — 7 sec] "Lien en bio" ou "Je montre tout dans la vidéo complète"
```

**REEL 60s :**
```
[HOOK — 5 sec] Résultat ou problème. Accroche immédiate.
[CONTEXTE — 10 sec] Une phrase : "J'avais ce besoin..."
[DÉMO — 35 sec] 3-4 étapes. Screen recording commenté.
[RÉSULTAT — 5 sec] Montre le produit fini.
[CTA — 5 sec] Abonnement ou lien bio.
```

Chaque ligne du script = ce qu'Edouard dit à voix haute.
Les indications de tournage entre crochets `[comme ça]`.
`[SCREEN]` = montre l'écran. `[FACE CAM]` = face caméra.

### 🎬 Notes de tournage
2-3 lignes max : ce qu'il faut préparer avant de filmer (écran ouvert sur quoi, ambiance, etc.)

### 📊 Angle utilisé
Nom de la catégorie de hook (ex: "Résultat choc") — pour enrichir la hooks-bank.

## Étape 4 — Notion

Après avoir livré le script, crée une entrée dans le Calendrier Notion (base `📅 Calendrier de contenu`) avec :

**Propriétés :**
- **Titre** : le hook principal (première phrase)
- **Statut** : Prêt à tourner
- **Format** : Reel 30s ou Reel 60s selon le script
- **Plateforme** : Instagram Reel
- **Hook principal** : le hook adapté
- **Lien vidéo** : l'URL de la vidéo source originale
- **Funnel** : toujours "🔝 Top of Funnel" pour l'instant (focus acquisition)

**Corps de la page (content) :**
Inclure dans le corps de la page, dans cet ordre :

```
## 🔗 Vidéo source
[URL de la vidéo originale — toujours inclure]

## 🎬 Ce que montre la vidéo
Description visuelle précise de la vidéo originale :
ce qui se passe à l'écran, ce que fait le créateur,
comment il interagit avec l'audience, quel est le CTA exact.
(ex: "Montre son dashboard avec ses agents IA, liste les rôles un par un,
CTA engagement : 'commentez GROW pour recevoir la méthode'")

## 📐 Angle & stratégie
Pourquoi cette vidéo marche — le mécanisme derrière.
(ex: "Résultat choc + CTA engagement pour booster l'algo.
La liste de titres corporate crée de la curiosité : qui fait tout ça ?")

## 🎣 Hook adapté
[la première phrase prête à dire, en français parlé]

## 📋 Script complet
[le script avec toutes les indications de tournage]

## 🎬 Notes de tournage
[ce qu'il faut préparer avant de filmer — 2-3 lignes max]
```

Confirme à Edouard : "✅ Ajouté dans ton Calendrier Notion — script complet + contexte inclus."
