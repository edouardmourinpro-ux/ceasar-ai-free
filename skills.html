des Skills Claude Code

Les skills sont des apps pour ton IA. Tu installes une skill, Claude apprend une nouvelle capacité. Voilà celles qui valent vraiment le coup.

▸ C'est quoi une skill ?

Une skill = un fichier texte avec des instructions. Tu le déposes dans un dossier, Claude le lit, il sait quoi faire. Pas de téléchargement, pas de package manager.

Globale (tous tes projets)

~/.claude/commands/nom-skill.md

Locale (un projet uniquement)

.claude/commands/nom-skill.md

Skills officielles Anthropic

Pré-installées, 100% sûres, 100% gratuites.

/batch

La skill de Garry Tan CEO de Y Combinator

Garry Tan, CEO de Y Combinator — le programme qui a lancé Airbnb, Dropbox et Stripe — a publié sa liste personnelle de skills Claude Code. gstack est sa principale : un navigateur headless qui permet à Claude de tester et d'inspecter n'importe quel site web directement depuis le terminal.

gstack PAR GARRY TAN · YC

Un navigateur headless intégré dans Claude Code. Tu lui donnes une URL, il ouvre la page, clique, remplit des formulaires, prend des screenshots et te rapporte ce qu'il voit. Plus besoin de tester manuellement.

Commande d'installation

$ git clone --single-branch --depth 1 https://github.com/garrytan/gstack.git ~/.claude/skills/gstack && cd ~/.claude/skills/gstack && ./setup

⚠ Avant d'installer une skill communautaire

Une étude Snyk (fév. 2026) a scanné 3 984 skills communautaires. Résultat : 36% contenaient des attaques par injection de prompt. Une skill malveillante peut lire tes fichiers, voler tes clés API ou exécuter des commandes cachées.

  • → Reste sur les skills officielles Anthropic en priorité
  • → Lis toujours le fichier .md avant d'installer
  • → Méfie-toi si une skill demande Bash(*) sans raison claire
