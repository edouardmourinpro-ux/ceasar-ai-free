# CLAUDE.md — Mon Projet

## Contexte
Je construis [ton projet] pour [ton audience].
Objectif : [ce que tu veux accomplir].

## Ma Voix
- Ton : direct, concret, pas de jargon
- Langue : français
- Style : montrer des résultats réels, pas de théorie

## Mon Audience
[Décris en 2-3 lignes qui tu cibles]
Ex : entrepreneurs français curieux de l'IA, non-devs

## Skills Actives
- /hook-gen → génère 10 hooks viraux pour ma niche
- /script-fr → écrit un script YouTube/Reel complet
- /repro → reproduit une vidéo virale avec mon angle

## Outils Connectés
- Notion : base de contenu + calendrier
- GitHub : cours et skills publiques
- Skool : communauté et cours

## Ce que Claude ne fait pas
- Ne pas inventer de stats ou témoignages
- Ne pas utiliser un anglicisme quand il existe un mot FR
- Toujours demander avant de modifier un fichier critique

# Fin du CLAUDE.md

5 minutes.

Copie le template ci-dessus et adapte-le avec tes infos (projet, audience, voix).

3

Lance Claude Code dans ce dossier. Il lira automatiquement ton CLAUDE.md au démarrage.
