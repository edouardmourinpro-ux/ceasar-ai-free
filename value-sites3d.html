Workflow complet : image Gemini → vidéo Kling → frames WEBP → site cinématique → déploiement Vercel. Sans coder une seule ligne.

Le workflow

4 étapes.
45 minutes.

Pas de framework. Pas de terminal. Claude fait tout. Tu fournis l'idée.

1

Image avec Gemini 5 min

Tu décris le visuel de ton site à Gemini. Il génère une image de départ à haute qualité — le décor, l'ambiance, les couleurs. C'est le point de départ cinématique.

Gemini Image Generation
2

Vidéo avec Kling 10 min

L'image devient une vidéo animée avec Kling AI. Un mouvement de caméra lent, cinématique — comme un dolly shot sur fond de ton produit ou de ton concept.

Kling AI · Image to Video
3

Frames WEBP avec Claude Code 5 min

Claude Code extrait les frames de la vidéo et les convertit en WEBP optimisés. Il génère aussi le HTML/CSS/JS du site scroll-driven entier.

/ceasar:build-3d-site
4

Déploiement Vercel 2 min

Une commande. Le site est en ligne sur Vercel avec une URL publique. Prêt à partager, prêt à convertir.

▲ vercel deploy
